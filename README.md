# OverDnD

Десктопное оверлей-приложение для мастеров Dungeons & Dragons. Компактный набор мини-приложений, которые всегда под рукой поверх любого окна -- для ведения боёв, бросков кубиков, управления музыкой и заметок прямо во время игровой сессии.

## Возможности

![explorer_oR0faFAM1g](https://github.com/user-attachments/assets/b78c4bbd-f230-4a3d-9013-65812d75c554)

### Инициатива
Трекер порядка хода в бою. Добавление игроков и существ с HP, классом доспеха и модификатором инициативы. Автоматический бросок и сортировка. Журнал боя с таймстампами событий.

### Пресеты существ
Библиотека сохранённых шаблонов существ (имя, модификатор инициативы, HP, КД) для быстрого добавления в бой.

### Статы
Таблица навыков игроков с модификаторами. Броски d20 по навыкам с историей результатов. Подсветка критических успехов и провалов.

### Управление игроками
Добавление, переименование и удаление игроков. Настройка модификаторов навыков для каждого игрока.

### Кубы
Бросок стандартных костей D&D (d4, d6, d8, d10, d12, d20, d100) с настраиваемым количеством и модификатором. История бросков.

### Звуки
Саундборд для быстрого воспроизведения звуковых эффектов из папки `Audio/`. Регулировка громкости.

### Музыка
Полноценный музыкальный плеер с поддержкой локальных файлов и YouTube. Несколько деков для организации плейлистов. Воспроизведение через LibVLC.

### Блокнот
Заметки с автосохранением. Шрифт Consolas для удобного форматирования.

### Настройки
- **9 тем оформления**: Тёмная, Светлая, Dracula, Solarized Dark, Лесная, Cyberpunk, Nord, Голубая (XP), Розовая
- **Акцентные цвета** для кастомизации интерфейса
- **3 масштаба UI**: Компактный (-5%), Стандартный, Крупный (+5%)

## Общие возможности интерфейса

- Безрамочные полупрозрачные окна в стиле оверлея
- Закрепление поверх всех окон (кнопка-пин)
- Запоминание позиций и размеров окон между сессиями
- Перетаскивание окон мышью
- Защита от запуска второй копии приложения

## Технологии

- **WPF** (.NET 5.0, C#)
- **NAudio** / **NAudio.Vorbis** -- воспроизведение аудио
- **LibVLCSharp** -- музыкальный плеер
- **WebView2** -- интеграция с YouTube
- **TagLibSharp** -- метаданные аудиофайлов

## Сборка и запуск

### Требования
- .NET 5.0 SDK
- Windows 10/11

### Debug
```bash
cd DnDOverlay.App
dotnet run
```

### Release (self-contained, single-file)
```bash
cd DnDOverlay.App
dotnet publish -c Release -r win-x64 --self-contained true
```

Собранный исполняемый файл `OverDnD.exe` появится в `bin/Release/net5.0-windows/win-x64/publish/`.

## Структура проекта

```
DnDOverlay.App/
  MainWindow.xaml            -- главное меню (панель с кнопками)
  MiniApps/
    InitiativeMiniAppWindow  -- трекер инициативы
    StatsMiniAppWindow       -- статы и броски навыков
    DiceMiniAppWindow        -- кубы
    SoundsMiniAppWindow      -- саундборд
    MusicMiniAppWindow       -- музыкальный плеер
    PlayerManagementWindow   -- управление игроками
    CreaturePresetsWindow    -- пресеты существ
    BattleLogWindow          -- журнал боя
    DiceHistoryWindow        -- история бросков
    AppSettingsWindow        -- настройки
  NotesMiniAppWindow.xaml    -- блокнот
  Infrastructure/            -- менеджеры данных, пути, утилиты
  Theming/                   -- темы и акцентные цвета
  Styles.xaml                -- общие стили
Audio/                       -- звуковые эффекты (.ogg)
Music/                       -- музыкальные файлы
Data/                        -- сохранённые данные (JSON)
```
